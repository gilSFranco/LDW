# API Rest com Flask - The Games

## Criar o ambiente virtual
```bash
python -m venv venv
```

## Ativar o ambiente virtual
```bash
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

```bash
venv/Scripts/activate
```
OU
```bash
cd venv
cd Scripts
activate
```

## Desativando o ambiente virtual
```bash
deactivate
```
